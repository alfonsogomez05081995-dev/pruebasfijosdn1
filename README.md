# FijosDN - Sistema de Gestión de Activos Fijos

Este proyecto es una aplicación web construida con **Next.js** y **Firebase**, diseñada para la gestión integral de activos fijos (asignación, inventario, logística).

## Estructura del Proyecto

- **Frontend:** Next.js (App Router)
- **Backend/Servicios:** Firebase (Firestore, Auth, Functions, Storage)
- **Estilos:** Tailwind CSS + Componentes UI (Shadcn/ui)

## Primeros Pasos

Para comenzar a trabajar en el proyecto:

1.  Instala las dependencias:
    ```bash
    npm install
    ```

2.  Ejecuta el servidor de desarrollo:
    ```bash
    npm run dev
    ```

3.  Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
